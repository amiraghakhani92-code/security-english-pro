# Access English Pro

A workplace-English training app for Access Control, Badging, Identity Management, GSOC reporting, radio communication, phone calls, meetings, emails, and job interview preparation.

## Mission

This is not a general English app.

It is designed to help Amir prepare for high-paying Access Management Specialist roles in Silicon Valley.

Target jobs: $50–60/hour Access Control, Badging, Identity Management, and Physical Security roles.

## Main Modules

- Dashboard
- Daily lessons
- GSOC report writing
- Radio communication
- Phone call practice
- Meeting English
- Email writing
- Access Control vocabulary
- Interview practice
- Progress tracker

## How to run

```bash
npm install
npm run dev
```

## How to build

```bash
npm run build
```
